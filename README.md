# Metal Detecting Car

Course Project for **TA212 (Manufacturing Lab)**  
Instructor: **Prof. Arvind Kumar**  
Indian Institute of Technology Kanpur  

Duration: **Jan 2025 – May 2025**

---

## Overview

This project involved designing and building a small metal-detecting car capable of sensing metallic objects and alerting the user using a buzzer.

The system integrates mechanical design, fabrication, and embedded electronics into a working prototype.

---

## Features

- Detects nearby metal objects and activates a **buzzer alert**
- **Arduino UNO** used for motor and detection control
- Custom chassis designed using **Fusion 360** and **Autodesk Inventor**
- Fully fabricated prototype using standard manufacturing processes
- Controlled propulsion and steering

---

## Mechanical Design

The car chassis and components were modeled using:

- Autodesk Fusion 360
- Autodesk Inventor

Fabrication methods used:

- Lathe machining  
- Milling  
- 3D printing  

CAD models and assemblies are available in the `cad/` directory.

---

## Electronics and Control

An **Arduino UNO** is used to:

- Control DC motors for propulsion
- Control steering mechanism
- Read the metal detection sensor
- Activate a buzzer when metal is detected

The Arduino program is located in the `code/` directory.
